Ferrum theme from LunarVim for Waybar.

## Usage

Download `ferrum.css`, place the file into your Waybar configuration folder, typically `~/.config/waybar`, and include it in your `style.css`:

```
@import "ferrum.css";
```

## Example

```
* {
	color: @fg;
}

window#waybar {
	background-color: @bg;
}
```
